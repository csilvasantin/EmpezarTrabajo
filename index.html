EmpezarTrabajo

Arranque rapido de sesion

Script de inicio que prepara el entorno de trabajo de forma consistente: comprueba GitHub CLI, asegura autenticacion, clona o actualiza onboarding y sincroniza los repos habituales.

Evita que cada sesion empiece desde cero.

  1. Ejecutar ./start_work.sh
  2. Abrir onboarding y pedir haz onboarding
  3. Continuar con el repo de trabajo activo
